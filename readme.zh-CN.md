# 轻松环世界（EasyRim）

[English](readme.md)

EasyRim 是一个刻意追求超强效果的《环世界》功能模组，聚焦殖民者强化、生产提速和世界地图机动性。

> **警告：** 本模组会显著降低游戏难度。

## 核心强化系统

- **超凡融合：** 提供认知强化、身体强化、生物强化、环境适应和灵能强化，并包含适应性装甲和永久生效的纳米作战协议。
- **紧急纳米重构：** 超凡融合的生物强化分支可阻止死亡，并在殖民者重伤倒下后延时完成重构。
- **不朽之魂：** 提供阶段化的生命与战斗增益，后期拥有极强的再生和近战压制力。
- **血原质共鸣：** 随血原质储量在鼎盛、充沛、均衡、不足和匮乏阶段间动态变化，同时获得对应的增益或减益。
- **先驱意志：** 全面强化工作、搬运、射击、恢复和生存能力。

## 能力系统

- **赤钢融血：** 短时间降低受到的伤害。
- **机械复原：** 快速治疗严重伤势并恢复失血。
- **血肉雕塑师：** 将尸体改造成不受控制的血肉怪兽。
- **血肉溶解：** 以毁灭性的灵能攻击处决活体目标。
- **吸血宇宙：** 可汲取任何符合条件的活体目标，包括敌对单位，并获得更多血原质。
- **机械复生：** 无需消耗资源即可复活。
- **纳米解析传送：** 快速移动，并支持选择世界地图目标。
- **纳米中继投送：** 召来多个一次性打击投送舱。
- **工作指导：** 暂时为目标提供最高 10 级的有效技能加成，并将学习速度提高至三倍。
- **殖民地动员：** 为地图上符合条件的殖民者提供全局心情和工作速度增益。

## 建筑与生产

- **自动钻探机：** 无需操作员即可开采深层资源；以太阳能驱动，资源耗尽后自动关机，并可自由重新部署。
- **可建造生命支持单元：** 保持房间的温度和压力，同时提供大量电力。
- **批量制造组件：** 一次性批量制造工业组件和高级组件。
- **配制中性胺：** 使用化学燃料、草药和精神叶片配制中性胺。
- **纺织材料：** 编织合成纤维并铸造超织物。

## 系统提速与平衡调整

- 大幅缩短塑形舱的伤口治疗、生体递归、年龄逆转和愉悦调整周期。
- 将无限反应釜调整为更快的固定产出节奏。
- 加快垃圾分解器的处理速度，并提高设备内部容量。
- 大幅提高地热发电机和飞船反应堆的发电量。
- 为玩家控制的掘进机增加深层资源探测能力。
- 调整“莽夫”特性的近战取舍。

当前角色强化主要通过 Hediff 与 Ability 系统实现；`TraitDef` 暂无实际玩法条目。

## 兼容性

- RimWorld 1.6（Odyssey）

# EasyRim / 轻松环世界

## 中文
这是一个让环世界更轻松的功能增强模组，聚焦于殖民者强化与玩法提速。

主要内容项：
- 殖民者特性
- 能力系统（战斗、治疗、复活、传送、动员等）
- 状态增强（超越系列、自适应护甲、狂战士之血、血液血红素等）
- 心情效果
- 合成配方（含中性胺与批量组件）
- 建筑内容
- 平衡补丁
- 中英本地化

警告：这是超强化模组，会显著降低游戏难度。

## English
This is a gameplay-easing enhancement mod for RimWorld, focused on pawn power-ups and faster progression.

Content includes:
- Pawn traits
- Ability system (combat, healing, resurrection, teleportation, rally)
- Hediff-based buffs (Transcendence set, Adaptive Armor, Berserker Blood, Blood Hemogen, etc.)
- Mood effects
- Crafting recipes (including neutroamine and batch components)
- Building content
- Balance patches
- Chinese and English localization

Warning: This is an overpowered mod and can make the game much easier.

Supported version: RimWorld 1.6 (Odyssey)
